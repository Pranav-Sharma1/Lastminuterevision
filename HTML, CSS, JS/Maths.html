Key Formulas:

Area of shapes:

Triangle = 1/2 base x height.
Parallelogram = base x perpendicular height.
Circle = πr²
Trapezium = 1/2 the sum of the parallel sides x perpendicular
distance in-between.
Rhombus = (Length x Height)/2
Sphere = 4/3 x πr²
Prism = Cross-sectional area x length.

Expanding brackets:

(a + b)² = a² + b² + 2ab
(a − b)² = a² + b² − 2ab.
(a − b)(a + b) = a² − b²
(x + a) (x + b) = x² + (a + b) x + ab.
(a + b + c) 2 = a² + b² + c² + 2ab + 2bc + 2ca

Index laws:

a¹ = a
aᵃ x aᵇ = aᵃ ⁺ ᵇ
aᵃ ÷ aᵇ = aᵃ ⁻ ᵇ
(aᵇ)ᵃ = aᵇᵃ
a⁰ = 1
a⁻ᵇ = 1/aᵇ

Pythagoras theorem

Hypotenuse: a² + b² =c²

Trigonometry:

SOH is Sin = Opposite / Hypotenuse
CAH is Cos = Adjacent / Hypotenuse
TOA is Tan = Opposite / Adjacent

Angles:

Corresponing angles or F angles are equal.
Alternate angles or Z angles are also equal.
Opposite angles are equal.
Angles on a straight line equal to 180 degrees.
Interior angles = 180(n-2).
Exterior angles = 360/n.
Angles around a point = 360 degrees.
